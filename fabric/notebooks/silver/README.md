# Silver Layer

## Overview

The Silver layer transforms raw aviation data into cleansed, standardized, and business-ready datasets.

## Objective

- Improve data quality
- Standardize formats and data types
- Apply business rules
- Prepare data for dimensional modeling

## Data Processing

- Null handling
- Data type standardization
- Date conversions
- Duplicate removal
- Data quality validation
- Reference data enrichment

## Silver Tables

- silver_flights
- silver_airports
- silver_airlines
- silver_routes

## Technologies

- Microsoft Fabric Lakehouse
- PySpark
- Delta Tables

## Output

Trusted and standardized datasets used to build dimensions, fact tables, and business marts.
